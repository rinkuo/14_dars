# 📚 Maktab CRM - School Management System 🎓

Welcome to the **Maktab CRM**, a powerful and user-friendly School Management System! This project helps manage students, teachers, groups, subjects, and more in a modern web application. Designed to simplify administration tasks and enhance the learning experience. 💻✨

---

## 🌟 Features

- **👩‍🏫 Teacher Management**: Add, update, and delete teacher records.
- **🧑‍🎓 Student Management**: Add new students, view their profiles, and track their progress.
- **📚 Group Management**: Create and manage student groups for each class.
- **📅 Timetable Integration**: Easily assign subjects and teachers to different timeslots.
- **🔍 Search and Filter**: Quickly find students, teachers, or groups with the powerful search functionality.
- **🛠️ Easy to use Interface**: A simple and intuitive UI for school admins and teachers.

## 💡 Installation

To get started with the **Maktab CRM** on your local machine, follow these steps:

### 1. Clone the repository:
```bash
git clone https://github.com/your-username/maktab-crm.git
```

### 2. Install dependencies:
```bash
cd maktab-crm
pip install -r requirements.txt
```

### 3. Set up the database:
```bash
python manage.py migrate
```

### 4. Run the development server:
```bash
python manage.py runserver
```

Now, you can access the application by going to [http://127.0.0.1:8000](http://127.0.0.1:8000) in your browser. 🌐
